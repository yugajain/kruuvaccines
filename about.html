This website was an attempt to provide for the community by making a vaccination tracking application. The application is not yet implemented with a server side application or a database stopping the application from reaching its full potential. If I decide to take this application any further, I will go deeper into the server side application completing the app. This app is still somewhat near a prototype because it still has to be implemented with the actual vaccine names and the full list of cities that may be easily derived from the help of api's. Also, this about page is somewhat barebone for which I apologize.

This app has the ability to select a specific city and list the vaccines available in the city. An application of this kind can help the community a ton.
